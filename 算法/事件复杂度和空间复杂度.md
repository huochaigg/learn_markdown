常见时间复杂度（从快到慢）：

| 时间复杂度      | 举例            |
| ---------- | ------------- |
| O(1)       | 常数时间          |
| O(log n)   | 二分查找          |
| O(n)       | 遍历数组          |
| O(n log n) | 快速排序、归并排序     |
| O(n²)      | 双重循环（冒泡、选择排序） |
| O(2ⁿ)      | 递归（斐波那契、子集枚举） |
| O(n!)      | 全排列           |

空间复杂度（Space Complexity）
空间复杂度是指**算法运行过程中占用额外内存空间的大小**（不包括输入数据本身）。
